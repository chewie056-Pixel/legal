# Politique de confidentialité - FrigoMalin

**Date d'entrée en vigueur : 5 janvier 2026**

## Introduction

FrigoMalin ("nous", "notre", "nos") respecte votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile FrigoMalin.

## Informations que nous collectons

### Informations fournies par l'utilisateur

L'application FrigoMalin stocke localement sur votre appareil :

- Les codes-barres des produits que vous scannez
- Les noms et marques de produits
- Les dates de péremption (DLC/DLUO)
- Les photos des dates de péremption que vous capturez
- Les informations nutritionnelles provenant d'OpenFoodFacts

**Toutes ces données sont stockées UNIQUEMENT sur votre appareil et ne sont JAMAIS envoyées à nos serveurs.**

### Informations collectées automatiquement

- **Données publicitaires** : Google AdMob peut collecter des données anonymes pour afficher des publicités personnalisées (ID publicitaire, interactions avec les annonces).
- **Données d'utilisation** : Des statistiques d'utilisation anonymes peuvent être collectées pour améliorer l'application.

## Comment nous utilisons vos informations

Les données stockées localement sont utilisées pour :

- Gérer votre inventaire alimentaire personnel
- Vous alerter sur les produits proches de leur date de péremption
- Afficher les informations nutritionnelles

Les données publicitaires sont utilisées par Google AdMob pour :

- Afficher des publicités pertinentes
- Mesurer l'efficacité des publicités
- Générer des revenus pour maintenir l'application gratuite

## Partage des données

Nous ne partageons, ne vendons ni ne louons vos données personnelles.

Les seules données partagées sont :

- **Google AdMob** : Pour afficher des publicités (données anonymes uniquement)
- **OpenFoodFacts** : Uniquement les codes-barres des produits pour récupérer les informations nutritionnelles publiques

## Permissions de l'application

FrigoMalin demande les permissions suivantes :

- **Caméra** : Pour scanner les codes-barres et photographier les dates de péremption
- **Internet** : Pour :
  - Récupérer les informations nutritionnelles depuis OpenFoodFacts
  - Afficher les publicités AdMob

## Stockage des données

Toutes vos données personnelles (inventaire, photos, dates) sont stockées **uniquement sur votre appareil** dans une base de données SQLite locale.

Si vous désinstallez l'application, toutes vos données seront définitivement supprimées.

## Services tiers

### OpenFoodFacts

- Service open-source et gratuit
- Politique de confidentialité : https://world.openfoodfacts.org/privacy
- Nous envoyons uniquement les codes-barres pour récupérer des informations publiques

### Google AdMob

- Service publicitaire de Google
- Politique de confidentialité : https://policies.google.com/privacy
- Politique publicitaire : https://policies.google.com/technologies/ads

### Google ML Kit

- Service d'OCR (reconnaissance de texte) qui fonctionne **hors ligne**
- Aucune donnée n'est envoyée à Google
- Les photos sont traitées localement sur votre appareil

## Sécurité

Nous prenons la sécurité de vos données au sérieux :

- Données stockées localement sur votre appareil
- Pas de transmission de données personnelles à nos serveurs
- Connexions sécurisées (HTTPS) pour OpenFoodFacts et AdMob

## Vos droits

Vous avez le droit de :

- **Accéder** à vos données : Elles sont visibles dans l'application
- **Modifier** vos données : Via l'interface de l'application
- **Supprimer** vos données : En désinstallant l'application ou en supprimant les produits individuellement
- **Refuser les publicités personnalisées** : Via les paramètres de votre appareil

### Désactiver les publicités personnalisées

**Android :**

1. Ouvrez Paramètres > Google > Annonces
2. Activez "Désactiver la personnalisation des annonces"

**iOS :**

1. Ouvrez Réglages > Confidentialité > Publicité
2. Activez "Suivi publicitaire limité"

## Protection des mineurs

FrigoMalin ne collecte pas sciemment d'informations personnelles d'enfants de moins de 13 ans. L'application ne nécessite aucune inscription ni création de compte.

## Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Les modifications seront publiées dans cette page avec une nouvelle date d'entrée en vigueur.

Nous vous encourageons à consulter régulièrement cette politique.

- Aucun transfert de données hors UE
- Droit d'accès, de rectification et d'effacement respectés
- Base légale : Consentement et intérêt légitime (publicités)
